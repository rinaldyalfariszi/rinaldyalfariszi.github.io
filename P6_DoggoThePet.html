Project Hand Tracking Game Doggo The Pet

A hand tracking game designed for the demo of the hand tracking device Ultraleap 3Di, where the objective is to move the ball into the dog's bowl to score points.

Status : Finished

Platform : Desktop

Technology : Ultraleap 3Di Hand Tracker, Unity Game Engine, Blender 3D

Demo video of Doggo The Pet

I was assigned to this project by my lecturer to develop a demo game for the Ultraleap 3Di device. The game will later be used in Telkom University's 32nd Anniversary Event. It's my first game that uses hand tracking with the Ultraleap 3Di device. I've also made modifications so that the game can be played without the hand tracking feature.

Core Mechanics

The gameplay of Doggo The Pet

The core mechanic is very simple, you just have to move the ball into the dog's bowl to score points. If you want to reset the game, simply press or click the Red Button or the 'Space' key on the keyboard.

What have I worked on?

  • Utilized Ultraleap 3Di device and Ultraleap SDK to implement the hand tracking mechanic.

  • Developed the core mechanic

  • Created the 3D visual elements assets, such as characters, objects, and environments, for the game.
