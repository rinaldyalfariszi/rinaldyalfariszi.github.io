Project Mobile Game Cowboy Rush

A first-person shooter and third-person shooter game that share the same objective, which is to score points by shooting the target within a limited time.

Status : Finished

Platform : Android

Technology : Unity Game Engine, Blender 3D

Demo game Cowboy Rush FPS Version

Demo game Cowboy Rush TPS Version

This is the first game I made with C# script. The game was created as an assignment for my college subject. The game is made in 2 versions, the first one is a First-Person Shooter (FPS), and the second one is a Third-Person Shooter (TPS).

Core Mechanics

The gameplay of game Cowboy Rush TPS and FPS

The main objective is to shoot the targets and reach a score of 100 points within a limited time using the provided weapon. If you can't do that, then you lose.

What have I worked on?

  • Integrated mobile player controller using joystick pack asset

  • Developed shooting and scoring mechanics

  • Integrated 3d external assets to be used in the game
