Project Mobile Game Cubethon Online

Inspired by Brackeys's Unity game tutorial, this hyper-casual mobile game challenges player to reach the end of each level by avoiding obstacles, and it requires an online account to play.

Status : Finished

Platform : Android

Technology : Firebase, Unity Game Engine

Demo video of Cubethon Online

This project was created for a college subject called Interactive Website Programming. At that time, I didn't know much about it, so I decided to create a game by following Brackeys' Unity tutorial for beginners. Since it was a college project related to websites, I set up a Firebase server to store data from users who log in to the game. But so far, this is the game i loved the most because of its visual!

Core Mechanics

The gameplay of Virals Play - Count Me

This game has 3 level, each level has different color, basically the player just need to reach the end of each level by avoiding obstacles, thats all. pretty casual right :/
