Project Hand Tracking Application Hestia Product Launch Leapmotion Application

A hand tracking application that's used to officially introduce and launch Hestia Smart Lamp by the Rector of Telkom University.

Status : Finished

Platform : Desktop

Technology : Ultraleap 3Di Hand Tracker, Unity Game Engine, Blender 3D

Video of Hestia Smart Lamp Product Launch with Ultraleap 3Di

This project is actually pretty simple, but it's also quite important because we're developing an interactive application that the Rector of Telkom University will be using during Telkom University's 32nd Anniversary event, as part of the grand launch of the Hestia smart lamp product to the public.

My lecturer, who is the Head of Metalabs, picked me to join the team because I have experience with a similar hand tracking project using the Ultraleap 3Di device. So, I'm here to help develop the application and make sure everything runs smoothly for the big event.

Core Mechanics

The application is being used by the Rector of Telkom University

The core mechanic of the application is very simple, you just need to grab the lamp and place it in the designated spot. It also allows you to restart by pressing the 'R' key and automatically place objects by pressing the 'Space' key on the keyboard. These mechanics were intentionally designed to be simple because the application will be used by the Rector of Telkom University in a prestigious event, so I have to minimize any potential problems.

What have I worked on?

Testing the core mechanic with Ultraleap 3Di device

  • Utilized Ultraleap 3Di device and Ultraleap SDK to implement the hand tracking mechanic.

  • Implemented precautionary measures to address potential application failures

  • Developed the core mechanic

  • Created the 3D visual elements assets, such as objects and environments for the game.
